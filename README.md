## ✨ Features

- Basic operations: `+`, `−`, `×`, `÷`
- Decimal point support
- Keyboard input support
- Square root calculation `√`
- Percentage conversion `%`
- Memory functions:
  - `M+`: Add to memory
  - `M-`: Subtract from memory
  - `MR`: Recall memory
  - `MC`: Clear memory
- Error handling (e.g., divide by zero)
- Beautiful **dark glassmorphic theme**
- Fully responsive UI
